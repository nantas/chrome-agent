4.4 Account Link Status

This page presents the operations that users can perform on the three separate types of accounts (user accounts, network service accounts, and Nintendo Accounts), and describes how these operations change the links between these accounts.

Info:

For Switch consoles for China, substitute "account for China" wherever you see "Nintendo Account."

| Number | User Operation | Account Relationships | Description |
| --- | --- | --- | --- |
| 1 | Creating a User Account | ![Click to zoom in.](https://developer.nintendo.com/Account_Guide/contents/Attachments/Attach_159219887/163772729.png) | The state where the user has only created a user account on the Switch and can play games offline. In addition, the user may link this account with any Nintendo Account. At this point, the network service account does not exist. |
| 2 | Linking to a Nintendo Account | ![Click to zoom in.](https://developer.nintendo.com/Account_Guide/contents/Attachments/Attach_159219887/163772730.png) | The state where a Nintendo Account is linked to the user account. At this point, the system automatically creates a network service account and links it to the Nintendo Account. When the user is using network services like NPLN and the friends system, the state is assured to be this state (or the next state if the user has multiple Switch consoles). |
| 3 | Linking the Same Nintendo Account With Multiple Consoles | ![Click to zoom in.](https://developer.nintendo.com/Account_Guide/contents/Attachments/Attach_159219887/163772719.png) | The state where the same Nintendo Account is linked to user accounts on multiple Switch consoles. The user can use network services from multiple Switch consoles using the same network service account. The user account nickname and profile image are synchronized across multiple Switch consoles via the network service account. It is important to note that the relationship between the network service account and the game's save data on the Switch console differs depending on which console is connected to the network service. |
| 4 | Deleting a User Account | ![Click to zoom in.](https://developer.nintendo.com/Account_Guide/contents/Attachments/Attach_159219887/163772721.png) | When a user account is deleted, all data stored on the Switch console for that user account is removed. In addition, when the Switch console is initialized, all user accounts on that console are deleted. The relationship between the Nintendo Account and network service account on the server does not change at this point. By linking again to that same Nintendo Account, the user can continue to use that previously used network service account. However, save data deleted from the Switch console cannot be recovered unless it was backed up separately. |
| 5 | Deleting a Nintendo Account | ![Click to zoom in.](https://developer.nintendo.com/Account_Guide/contents/Attachments/Attach_159219887/163772722.png) | When a Nintendo Account is deleted, the associated network service account is also deleted.  If a user attempts to use network services from a Switch in this state, an error occurs. A user account in this state cannot be linked to a different Nintendo Account and network service account.  Info:  - The user account cannot be relinked to another Nintendo Account.  - There is currently no way for users to delete accounts for China. However, the accounts may be deleted via Nintendo actions. |
| 6 | Removing or Changing the Link Between a Network Service Account and a Nintendo Account | ![Click to zoom in.](https://developer.nintendo.com/Account_Guide/contents/Attachments/Attach_159219887/163772723.png) | After a network service account and Nintendo Account are linked, the link cannot be removed or changed by the user. However, the link can be removed or changed by certain special operations. |
| 7 | Unlinking a Nintendo Account | ![Click to zoom in.](https://developer.nintendo.com/Account_Guide/contents/Attachments/Attach_159219887/163772720.png) | Users cannot unlink Nintendo Accounts and network service accounts from user accounts. However, unlinking can be performed for reasons related to repair and customer support.  The relationship between the Nintendo Account and network service account on the server does not change at this point. The user can continue to use the network service account they had used previously by re-linking their user account with the same Nintendo Account.  Info:  The user account cannot be relinked to another Nintendo Account. |

For reference, the following list includes user operations that cannot be performed. The system assures that it will not transition to these states.

| Number | User Operations That Are Not Executable | Relationships | Description |
| --- | --- | --- | --- |
| 1 | Linking One Nintendo Account to Different User Accounts on the Same Switch | ![Click to zoom in.](https://developer.nintendo.com/Account_Guide/contents/Attachments/Attach_159219887/163772724.png) | There is a one-to-one relationship between user accounts and Nintendo Accounts (or network service accounts) on a single Switch console. It is not possible to link multiple user accounts on the one Switch to the same Nintendo Account. |
| 2 | Simultaneously Linking Multiple Nintendo Accounts or Network Service Accounts to One User Account | ![Click to zoom in.](https://developer.nintendo.com/Account_Guide/contents/Attachments/Attach_159219887/163772725.png) | There is a one-to-one relationship between user accounts and Nintendo Accounts (or network service accounts) on a single Switch console. It is not possible to link multiple Nintendo Accounts to one user account. |
| 3 | Linking Multiple Network Service Accounts to One Nintendo Account | ![Click to zoom in.](https://developer.nintendo.com/Account_Guide/contents/Attachments/Attach_159219887/163772726.png) | There is a one-to-one relationship between Nintendo Accounts and network service accounts. This relationship is never broken. |
| 4 | Linking One Network Service Account to Multiple Nintendo Accounts | ![Click to zoom in.](https://developer.nintendo.com/Account_Guide/contents/Attachments/Attach_159219887/163772727.png) | There is a one-to-one relationship between Nintendo Accounts and network service accounts. This relationship is never broken. |
| 5 | Deleting a Network Service Account | ![Click to zoom in.](https://developer.nintendo.com/Account_Guide/contents/Attachments/Attach_159219887/163772728.png) | It is not possible to delete just the network service account. The network service account is deleted only when the Nintendo Account to which it is linked is deleted.  Info:  There is currently no way for users to delete accounts for China. However, the accounts may be deleted via Nintendo actions. |
